Álgebra Linear

1º Grau: ax + b = 0 → x = -b/a

Ex: 2x + 4 = 0 → x = -2

Sistemas: Substituição/Eliminação/Addição

Geometria Plana

Triângulo: A = (base × altura)/2

Círculo: A = πr² | C = 2πr (π ≈ 3.14)

Teorema Pitágoras: a² + b² = c²

Trigonometria

Seno: oposto/hipotenusa | Cosseno: adjacente/hipotenusa

Triângulo 3-4-5: sin = 3/5 = 0.6

Identidades: sin² + cos² = 1

Funções

Linear: f(x) = ax + b | Quadrática: ax² + bx + c

Domínio/Imagem: Todos R / Vértice para cima/baixo

Raízes: Bhaskara Δ = b² - 4ac

Estatística

Média: Σxi/n | Mediana: meio ordenado

Desvio Padrão: √[Σ(xi - x̄)²/n]

Probabilidade: favoráveis/possíveis (0 a 1)
